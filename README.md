# CodeSlam

CodeSlam is a futuristic multiplayer competitive coding platform concept: LeetCode meets Valorant.

## Current state

- Frontend: React + TypeScript + Tailwind CSS + Framer Motion
- Backend scaffold: Spring Boot
- Focused first milestone: cinematic landing page for the battle arena experience

## Structure

- `frontend/` - Vite app with the landing page
- `backend/` - Spring Boot service scaffold for future real-time systems

## Next steps

- Run the frontend dev server
- Connect live matchmaking and WebSocket battle systems
- Add execution sandbox and multiplayer room logic
